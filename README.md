# repair-estimator
Программное обеспечение, которое по заданным размерам и форме помещений оценивает стоимость ремонта этого помещения. Результатом помимо вилки общей стоимости, дается полная смета в виде: какие позиции и в каком количестве надо закупать для выполнения ремонта и сколько стоят услуги каких специалистов.
