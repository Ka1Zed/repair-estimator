## Типы комнат и допустимые работы

Машиночитаемый источник правды — `docs/room-types.json`. Таблица ниже — для людей.

| Тип | Пол | Стены | Потолок | Плитка (доп.) | Сантехника |
|-----|-----|-------|---------|---------------|------------|
| Жилая (living) | ламинат / линолеум / паркет | покраска / обои | покраска / натяжной | нет | нет |
| Кухня (kitchen) | плитка / ламинат | покраска / плитка | покраска / натяжной | да | доступна |
| Санузел (bathroom) | только плитка | плитка / влагостойкая краска | влагостойкий / натяжной | да | обязательна |
| Коридор (hallway) | ламинат / плитка | покраска / обои | покраска / натяжной | нет | нет |

Электрика (basic / extended) доступна во всех типах.

Поле `tile` — отдельная работа по плитке (фартук на кухне, стены в санузле); пол плиткой задаётся через `floor: "tile"`.
